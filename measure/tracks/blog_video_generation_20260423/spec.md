# Specification: Blog-to-Video Generation Pipeline

## Overview
Build an automated pipeline that transforms Reading Advantage blog posts into TikTok-style 9:16 engagement videos (1-2 minutes). The pipeline uses MiniMax (`mmx` CLI) for generative assets (script, TTS audio, background images) and Revideo for programmatic video composition.

## Functional Requirements

### Video Structure
- **Intro (3s):** Reading Advantage logo on branded background with blog title.
- **Content (variable):** 4-6 dynamic segments, each showing:
  - A full-screen background image (generated by `mmx image` or blog cover image).
  - Large, bold text overlay with the segment's key message (rendered by Revideo).
  - Subtle Ken Burns zoom effect on the background image.
- **Outro (3s):** Branded CTA screen prompting viewers to read the full blog ("link in description").

### Input
- A blog markdown file path (e.g., `src/app/[locale]/(marketing)/blog/posts/th/future-education-ai.md`).

### Output
- A 9:16 MP4 video file saved to `public/videos/<slug>.mp4`.

### Pipeline Steps
1. **Parse:** Extract title, excerpt, tags, cover image, and body content from markdown frontmatter.
2. **Script Generation:** Use `mmx text chat` to summarize the blog into 4-6 punchy TikTok-style segments.
3. **Audio Generation:** Use `mmx speech synthesize` to generate narration MP3 from the full script.
4. **Image Generation:** Use `mmx image generate` (9:16 aspect ratio) to create a unique background image per segment.
5. **Video Composition:** Use Revideo (`@revideo/renderer`) to compose logo intro, content segments with text overlays, and CTA outro, synced to the narration audio.

## Non-Functional Requirements
- **Aspect Ratio:** 9:16 (1080x1920) for TikTok/Reels/Shorts.
- **Duration:** 60-90 seconds target.
- **Language Support:** Must work with Thai and English blog posts. TTS validated for Thai.
- **Fallbacks:** If AI script generation fails, fall back to paragraph-based segmentation. If image generation fails, fall back to the blog's cover image or logo.

## Acceptance Criteria
- [ ] Running `npx tsx scripts/generate-blog-video.ts <blog.md>` produces a valid MP4.
- [ ] The generated video includes the Reading Advantage logo at the start.
- [ ] The generated video includes a CTA at the end.
- [ ] Thai blog posts produce intelligible Thai TTS audio.
- [ ] Revideo project files are committed under `revideo/`.
