# CSS Utility Consolidation

## Problem
Overlapping CSS utilities (`.glass-morphism` vs `.hero-glass`) create confusion and maintenance burden.

## Solution
Merge `.glass-morphism` and `.hero-glass` into single `.glass` utility.

## Acceptance Criteria
- [ ] Single `.glass` utility class
- [ ] All references updated
- [ ] Visual appearance preserved
- [ ] No CSS regressions
